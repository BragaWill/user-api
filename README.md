User Api
====================

# Running locally
With Node installed, install the dependencies with:
```sh
$ npm i
```

and finally run:
```sh
nodemon index.js
```
or run: 
```sh
npx nodemon index.js
```

listening: 
```
http://localhost:6767/
```
